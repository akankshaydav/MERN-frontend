# Restaurant Reservation System

A **full-stack restaurant reservation system** built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js).  
This project allows users to view available tables, make reservations, and manage bookings.

---

## Features

- User-friendly interface to view and reserve tables
- Add, edit, and cancel reservations (if implemented)
- Backend API built with Node.js and Express
- MongoDB database for storing reservations and user data
- Responsive design using React.js

---

## Tech Stack

- **Frontend:** React.js, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ORM)  
- **Other Tools:** npm, dotenv, nodemon  

---
